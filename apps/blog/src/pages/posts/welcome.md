# Welcome to Minions Bundle: Portfolio

We're excited to introduce **Minions Bundle: Portfolio** — Top-level structural organizational bundle — categories, ecosystems, projects.

## What's Included

- **TypeScript SDK**: `npm install @minions-bundles-portfolio/sdk`
- **Python SDK**: `pip install minions-bundles-portfolio`
- **CLI**: `npm install -g @minions-bundles-portfolio/cli`
