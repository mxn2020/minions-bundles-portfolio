# Minions Bundle: Portfolio

Top-level structural organizational bundle — categories, ecosystems, projects

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@mxn2020/minions-bundles-portfolio-sdk`) — Core library for Node.js
- **Python SDK** (`minions-bundles-portfolio`) — Python bindings
- **CLI Tool** (`@mxn2020/minions-bundles-portfolio-cli`) — Command-line interface
- **Web App** — Interactive playground at [app.minions.wtf](https://app.minions.wtf)
