---
title: Installation
description: "Install Minions Bundle: Portfolio packages"
---

## TypeScript / Node.js

```bash
npm install @minions-bundles-portfolio/sdk minions-sdk
```

## Python

```bash
pip install minions-bundles-portfolio
```

## CLI

```bash
npm install -g @minions-bundles-portfolio/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
